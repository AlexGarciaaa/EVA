Tutorial de HTML i CSS

Estructura HTML

És una línia que conté informació sobre la versió de HTML, una capçalera (delimitada per l'element HEAD), i un cos amb el contingut de el document (delimitat per l'element BODY o FRAMESET).

Estructura CSS

A l'igual que els documents HTML, els documents CSS són arxius de text on s'escriu un seguit d'ordres i el client (navegador) les interpreta i aplica als documents HTML associats.

Etiquetes HTML

Podem fer servir les etiquetes HTML per crear paràgrafs de text (p), inserir imatges a la nostra pàgina web (img), crear taules (table), construir llistes (ul) (lit), ressaltar paraules o frases en negreta (strong) i un llarg etcètera.

Etiquetes CSS

En el full d'estil arxiu. css s'escriu com volem que sigui l'estil de presentació de les etiquetes color, mida, font, vores, marges, posició. A la pàgina web s'escriu quin full d'estil volem utilitzar.

Atributs HTML

Els atributs d'HTML són paraules especials utilitzades dins de l'etiqueta d'obertura, per controlar el comportament de l'element. Els atributs d'HTML són un modificador d'un tipus d'element d'HTML. En sintaxi HTML, un atribut s'afegeix a una etiqueta d'inici d'HTML.

Atributs CSS

Els atributs CSS són els que et permeten aplicar estils als elements HTML.

Regles HTML

Les regles de l'html són delimitar l'àmbit de les etiquetes (llevat d'excepcions), etiqueta interna es tanca abans que externa, etiquetes a nivell de bloc poden contenir etiquetes a nivell de línia, però no a l'inrevés, etiquetes no sensitives a majúscules, en comptes d'espais afegir ratlla el pis

Regles CSS

Està compost per parells propietat CSS i el seu corresponent valor. Propietat: característica que es modifica en l'element seleccionat, com ara la seva mida de lletra, color, etc. i cada propietat té un conjunt de possibles valors.

Propietats HTML

Les propietats d'HTML són utilitzades dins de l'etiqueta d'obertura, per controlar el comportament de l'element.

Propietats CSS

Les propietats abreujades són propietats CSS que permeten assignar el valor de moltes altres propietats de CSS a el mateix temps.

Comentaris HTML

Els comentaris html són textos que van dins d'el codi font però que no són mostrats pels navegadors. Aquests comentaris són molt útils per als editors de la pàgina, ja que ajuden a una major comprensió de el codi.

Comentaris CSS

Els comentaris són usats per afegir notes explicatorias o prevenir que el navegador interpreti parts del full d'estils. Els comentaris es poden posar en qualsevol espai en blanc que es permeti al full d'estils.

Valors HTML

Els valors se li poden assignar als elements, per exemple, a html se li pot assignar un atribut per indicar en quin idioma està escrit el document (lang) i podem assignar com a valor un codi d'idioma ( "és" per a l'espanyol, "en" per l'anglès)

Valors CSS

Un valor en CSS és una forma de definir una col·lecció de subvalores admès. Això vol dir que si veus (color) com a vàlid, no necessites preguntar quins dels diferents tipus de valor de color pots utilitzar: paraules clau, valors hexadecimals, funcions rgb ()
