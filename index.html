Tutorial de HTML y CSS

Estructura HTML

Es una línea que contiene información sobre la versión de HTML, Una cabecera (delimitada por el elemento HEAD), y un cuerpo con el contenido del documento (delimitado por el elemento BODY o FRAMESET).

Estructura CSS

Al igual que los documentos HTML, los documentos CSS son archivos de texto donde se escribe una serie de órdenes y el cliente (navegador) las interpreta y aplica a los documentos HTML asociados.

Etiquetas HTML

Podemos usar las etiquetas HTML para crear párrafos de texto (p), insertar imágenes en nuestra página web (img), crear tablas (table), construir listas (ul)(li), resaltar palabras o frases en negrita (strong) y un largo etcétera.

Etiquetas CSS

En la hoja de estilo archivo . css se escribe cómo queremos que sea el estilo de presentación de las etiquetas color, tamaño, fuente, bordes, márgenes, posición. En la página web se escribe qué hoja de estilo queremos utilizar.

Atributos HTML

Los atributos de HTML son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo de elemento de HTML. En sintaxis HTML, un atributo se añade a una etiqueta de inicio de HTML.

Atributos CSS

Los atributos CSS son los que te permiten aplicar estilos a los elementos HTML.

Reglas HTML

Los comentarios html son textos que dentro de el código fuente pero que no son mostrados por los navegadores. Estos comentarios son muy útiles para los editores de la página, ya que ayudan a una mayor comprensión del código.

Propiedades HTML

Las propiedades de HTML son utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento.

Propiedades CSS

Las propiedades abreviadas son propiedades CSS que permiten asignar el valor de muchas otras propiedades de CSS al mismo tiempo.

Comentarios HTML

Los comentarios html son textos que van dentro del código fuente pero que no son mostrados por los navegadores. Estos comentarios son muy útiles para los editores de la página, ya que ayudan a una mayor comprensión del código.

Comentarios CSS

Los comentarios son usados para añadir notas explicatorias o prevenir que el navegador interprete partes de la hoja de estilos. Los comentarios se pueden colocar en cualquier espacio en blanco que se permita en la hoja de estilos.

Valores HTML

Los valores se le pueden asignar a los elementos, por ejemplo, a html se le puede asignar un atributo para indicar en qué idioma está escrito el documento (lang) y podemos asignarle como valor un código de idioma ("es" para el español, "en" para el ingles)

Valores CSS

Un valor en CSS es una forma de definir una colección de subvalores admitido. Esto significa que si ves (color) como válido, no necesitas preguntarte cuáles de los diferentes tipos de valor de color puedes usar: palabras clave, valores hexadecimales, funciones rgb()
